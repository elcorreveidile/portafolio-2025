# Portafolio 2025 - Javier Benítez Laínez

Sitio web profesional con innovación tecnopedagógica, publicaciones y actividades literarias.

## Contenido

- **Innovación Tecnopedagógica**: Aula virtual Moodle, 3 chatbots IA, portales educativos con microsites
- **Proyectos**: 4 plataformas web educativas completas
- **Publicaciones**: Obra literaria y ponencias académicas
- **Actividades**: Residencias y lecturas literarias

## Despliegue en GitHub Pages

1. Crea repositorio `portafolio-2025`
2. Sube estos archivos
3. Activa GitHub Pages en Settings > Pages
4. Selecciona rama `main` y carpeta raíz
